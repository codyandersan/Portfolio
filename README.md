# Prakhar Aditya Tripathi — Portfolio

Personal portfolio site built with Next.js 15, Tailwind CSS, and TypeScript.

Live at [prakhartri.me](https://prakhartri.me)

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- pnpm

## Getting Started

Install dependencies:

pnpm install

Run the development server:

pnpm dev

Build for production:

pnpm build

Preview the production build:

pnpm start

## Project Structure

app/
  page.tsx       Main page
  layout.tsx     Root layout and metadata
  globals.css    Global styles and animations
public/
  profile.webp   Profile photo
  og.png         Open Graph image for link previews

## Features

- Typing animation on hero name
- Responsive layout for mobile and desktop
- Timeline of projects and milestones
- SEO optimized with Open Graph, Twitter cards, and JSON-LD structured data
- Devanagari script logo mark using Tiro Devanagari Hindi font
